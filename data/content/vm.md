В жизни порой возникают такие моменты, когда ваша ОС не удовлетворяет вашим требованиям (например, не хватает удобных приложений для работы с документами, нет подходящих редакторов, нет средств работы с языком программирования, который вы бы хотели изучить, неудобный графический интерфейс и т.п.). Неужели для этого нужно сносить свою текущую ОС и ставить другую (или делать dual boot)? На самом деле, перед тем, как делать что-то из этого, стоит дважды подумать, ведь другая ОС может не только дать вам что-то новое, но и что-то у вас отнять.

Для решения данной проблемы существуют виртуальные машины, которые позволяют "попробовать" другую ОС, либо полноценно работать с новой ОС, не удаляя текущей, к которой вы так привыкли. Вот некоторые из них: [Virtual PC](http://www.microsoft.com/ru-ru/download/details.aspx?..), [VMware](http://www.vmware.com/), [VirtualBox](https://www.virtualbox.org/), [Parallels Desktop](http://www.parallels.com/ru/products/desktop/).

Так же существует программа, позволяющая использовать другую ОС без графического интерфейса (в основном какую-нибудь Linux-based ОС) - [Vagrant](https://www.vagrantup.com/). Vagrant работает поверх виртуальной машины (например, VirtualBox) и устанавливает свои, специально подготовленные [образы из облака](https://vagrantcloud.com/), которые вы можете скачать и развернуть при помощи того же Vargant (по сути, виртуальная машина просто является хранилищем для Vagrant, поэтому её включать не обязательно).

**Тем, кто хочет поиграть с какой-нибудь Linux-based ОС, предлагаются следующие задания**:

* Установить VirtualBox
* Установить на VirtualBox [Xubuntu](http://xubuntu.org/)(либо какой-нибудь другой дистрибутив по желанию)
* Научиться работать с эмулятором терминала(основные команды, которые можно изучить: _man_, _cd_, _ls_, _cat_, _touch_)
