#### Зависимости
* [NodeJS](http://nodejs.org)
* [Gulp](http://gulpjs.com)

#### Как работать с исходниками
* Установить [NodeJS](http://nodejs.org) и проверить, что его можно запустить при помощи команды `node`
* Установить [Gulp](http://gulpjs.com) при помощи команды `npm install -g gulp`(возможно, данную команду нужно запустить от имени суперпользователя)
* Оцистить кэш NPM при помощи команды `npm cache clean`(возможно, данную команду нужно запустить от имени суперпользователя)
* В директории проекта выполнить команду `npm install`, которая установит локально необходимые для компиляции пакеты.
* Для того, чтобы скомпилировать исходники воспользуйтесь командой `gulp` в директории проекта.
* Для того, чтобы запустить проект на локальной машине, воспользуйтесь командой `gulp serve`. После скомпилированные искодники будут доступны по адресу `http://localhost:8080/`. При изменении исходников, они автоматически перекомпилируюся.

#### Структура
* `[source_dir]/dest` - директория, в которую попадает скомпилированный проект
* `[source_dir]/src` - директория, в которой нахотятся искодники шаблонов и стилей (`[source_dir]/src/template` и `[source_dir]/src/less` соответственно).
* `[source_dir]/data` - директория, в которой нахотятся: статические файлы(копируются в `[source_dir]/dest` без изменений), Markdown файлы с данными, файл конфигурации страниц проекта `[source_dir]/data/config.json`.
