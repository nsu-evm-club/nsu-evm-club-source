#### Зависимости
* [NodeJS](http://nodejs.org)
* [Grunt](http://gruntjs.com)

#### Как работать с исходниками
* В директории проекта выполнить команду `npm install`, которая установит локально необходимые для компиляции пакеты.
* Для того, чтобы скомпилировать исходники воспользуйтесь командой `grunt` в директории проекта (данная команда очищает директорию `[source_dir]/dest` и собирает проект в ней).
* Для того, чтобы запустить проект на локальной машине, воспользуйтесь командой `grunt serve`. После скомпилированные искодники будут доступны по адресу `http://localhost:9000/`.

#### Структура
* `[source_dir]/dest` - директория, в которую попадает скомпилированный проект
* `[source_dir]/src` - директория, в которой нахотятся искодники шаблонов и стилей (`[source_dir]/src/template` и `[source_dir]/src/less` соответственно).
* `[source_dir]/data` - директория, в которой нахотятся: статические файлы(копируются в `[source_dir]/dest` без изменений), Markdown файлы с данными, файл конфигурации страниц проекта `[source_dir]/data/config.json`.
